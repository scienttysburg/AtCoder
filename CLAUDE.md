# CLAUDE.md

このファイルは、このリポジトリで作業する Claude Code (claude.ai/code) へのガイダンスを提供します。

## 言語設定

- 常に日本語で会話する

## 概要

AtCoder の競技プログラミング問題に対する非公式の解答を複数言語でまとめるリポジトリです。また、[zenn-cli](https://zenn.dev/zenn/articles/zenn-cli-guide) を使って、解説記事を Zenn に投稿することも想定しています。

## Zenn CLI

記事は `articles/` ディレクトリに配置します。主なコマンドは以下の通りです。

```bash
npx zenn new:article          # 新しい記事を作成する
npx zenn preview              # ローカルでプレビュー (http://localhost:8000)
```

## リポジトリ構成

- `articles/` — Zenn 用の記事ファイル（解説・解法のまとめなど）
- 解答ファイルはコンテスト名・問題番号ごとに整理することを想定（解答が増えた時点で構成を確定する）

## 言語について

AtCoder は多くの言語に対応しています。解答を追加する際は、コンテスト名と問題番号で一貫したディレクトリ構成を取り、複数言語を使う場合は言語ごとにまとめるか、明確に区別できるよう命名してください。
